# AGENTS.md

## Tech Stack

- Frontend: React + TypeScript + Tailwind CSS
- Backend: FastAPI (Python)
- Database: Supabase (PostgreSQL)
- Deployment: Vercel / Railway

## Code Style

- Use TypeScript strict mode
- Follow Airbnb ESLint config
- Component files: PascalCase, hooks: camelCase with "use" prefix

## Testing

- Run `npm test` before finalizing any PR
- All API endpoints need at least one integration test

## PR Instructions

- Title format: [Feature/Fix/Refactor] Short description
- Always include "Testing Done" section in PR body
